{@linkplain org.opengis.metadata.citation.Citation Citation} and responsible party information. The following is adapted from OpenGIS® Metadata (Topic 11) specification.

This package of datatypes provides a standardized method ({@link org.opengis.metadata.citation.Citation}) for citing a resource (dataset, feature, source, publication, etc.), as well as information about the party responsible ({@link org.opengis.metadata.citation.ResponsibleParty}) for a resource. The {@link org.opengis.metadata.citation.ResponsibleParty} datatype contains the identity of person(s), and/or position, and/or organization(s) associated with the resource. The location ({@link org.opengis.metadata.citation.Address}) of the responsible person or organization is also defined here.
